# Windy Peak via Mountain Lion Loop

A concise hiking guide and personal log for the Windy Peak trail accessed via the Mountain Lion Loop in Colorado. This repository documents the hike details, gear planning, and personal observations to assist future trips and planning.

---

## Features

- Detailed hike outline including time, difficulty, and gear recommendations
- Personal gear review and lessons learned
- Links to external resources such as trail maps and park information

## Tech Stack

- Markdown for documentation
- Static site content format (index.md) presumably for a static site generator

## Getting Started

This repository contains documentation only; no software installation is required.

To view the content locally:

```bash
# Clone the repository
git clone https://github.com/justin-napolitano/windy-peak-mountain-lion-loop.git

# Open index.md in a markdown viewer or editor
code windy-peak-mountain-lion-loop/index.md
```
